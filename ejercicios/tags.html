Header

A header contains introductory content. We can have several, although we choose to have only one for the full document in this case.

Article

All of this document is inside an article tag. If we work more on the writing, we could interpret this full document as a single article describing less-known semantic elements. An article should be independent from the rest. We could apply article tags to every paragraph, but we would be violating the semantic restriction of article. Beware!

br, hr

br tag is a line break. There are several line breaks in this document. The next sections will be separated by hr. It is rendered as an horizontal line by default. However, the semantic information is more interesting: hr indicates a thematic break. Thus, we are supposed to talk about something else after hr tag in next source line.
